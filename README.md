## Getting Started

### Prerequisites

- Install [Node.js](https://nodejs.org/).
- Refer to the `.mock.env` file and create the `.env` & `.env.dev` file in the root directory.

### Run

development mode

```bash
yarn install
yarn dev
```

production mode

```bash
yarn install
yarn build
yarn start
```
